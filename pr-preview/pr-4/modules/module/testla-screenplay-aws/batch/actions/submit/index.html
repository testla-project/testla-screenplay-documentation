Submit

The Submit class provides a convenient way to submit a new batch job.

Extends​

This class extends the abstract Action class from Core.

Methods​

performAs​

public async performAs(actor: Actor): Promise<SubmitJobCommandOutput>;
  • Description: Submit a batch job.
  • Parameters:
    • actor - The actor performing the action.
  • Returns: Promise<SubmitJobCommandOutput> - The response object.

job​

Introduced in: 1.0.0

public static job(submitJobCommandInput): Submit;
  • Description: Create a new instance of the Submit class with the specified submit job command.
  • Parameters:
    • submitJobCommandInput - The SubmitJobCommandInput object.
  • Returns: Submit - A new instance of the Submit class.

Usage:

await actor.attemptsTo(
    Submit.job(submitJobCommandInput),
);

Methods inherited from Core Action​

orSkipOnFail​

Introduced in core: 0.5.0

public get orSkipOnFail(): Action;
  • Description: Allows to skip an action on fail.
  • Returns: Action - Returns the current action.

Usage:

// Would skip the step on error without breaking the execution
await actor.attemptsTo(
    Do.something().orSkipOnFail,
);

withAbilityAlias​

Introduced in core: 0.3.0

public withAbilityAlias(alias: string): Action;
  • Description: Defines the ability alias to be used during execution.
  • Parameters:
    • alias - The alias.
  • Returns: Action - Returns the current action.

Usage:

await actor.attemptsTo(
    Do.something().withAbilityAlias('myAlias'),
);

Introduction

Testla Screenplay Essentials is a collection of generic Actions and Questions which can be used in any context. It most likely is installed along with a module for a specific purpose like in example Testla Screenplay Playwright.

Installation

To install it, use the following command based on your package manager:

  • npm
  • Yarn
npm install @testla/screenplay-essentials
yarn add @testla/screenplay-essentials

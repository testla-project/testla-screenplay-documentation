# Testla Screenplay Modules

The following packages are available. You may want to include them into your test repository as per your projects needs.

## Testla Screenplay (Core)

The fundament of Testla Screenplay on which all other Testla Screenplay packages are built. It is essential for custom development of own Testla Screenplay packages in order to stay compatible with all other Testla Screenplay packages.

Repository: https://github.com/testla-project/testla-screenplay-core-js

NPM: https://www.npmjs.com/package/@testla/screenplay

## Create Testla Screenplay CLI Tool

A CLI tool offering a setup routine for Testla Screenplay allowing you to pick the needed packages during the install routine. It creates a full project folder structure with sample Screenplay elements like Actions and Questions so that you can right away start writing test code without going through a complicated manual setup phase.

Repository: https://github.com/testla-project/create-testla-screenplay

NPM: https://www.npmjs.com/package/@testla/create-screenplay

## Testla Screenplay Playwright

Implementation of Testla Screenplay for Playwright coming with Abilities, Actions and Questions translating into regular Playwright commands. The package holds custom reporters like HTML report which can be announced in the Playwright config.

Repository: https://github.com/testla-project/testla-screenplay-playwright-js

NPM: https://www.npmjs.com/package/@testla/screenplay-playwright

## Testla Screenplay Essentials

A collection of generic Actions and Questions which can be used in any context.

Repository: https://github.com/testla-project/testla-screenplay-essentials-js

NPM: https://www.npmjs.com/package/@testla/screenplay-essentials

## Testla Screenplay Assets

A collection of assets like logos and documentation which are used in other Testla Screenplay packages.

Repository: https://github.com/testla-project/testla-screenplay-assets

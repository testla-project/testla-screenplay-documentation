# Testla Screenplay (Core)

The fundament of Testla Screenplay on which all other Testla Screenplay packages are built. It is essential for custom development of own Testla Screenplay packages in order to stay compatible with all other Testla Screenplay packages.

Repository: https://github.com/testla-project/testla-screenplay-core-js

NPM: https://www.npmjs.com/package/@testla/screenplay

Introduction

BETAThis module is currently in beta state.

Screenplay AWS comes with Abilities, Actions and Questions to interact with AWS resources.

The package currently supports the following resources:

  • Batch
  • Kinesis
  • S3

Installation

To install it, use the following command based on your package manager:

  • npm
  • Yarn
npm install @testla/screenplay-aws
yarn add @testla/screenplay-aws
